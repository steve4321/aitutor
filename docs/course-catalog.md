|------|------|---------|
| 第一步 | 通读全诗，了解大意 | 通读1-2遍，用自己的话复述诗歌表面意思，找出写的是什么(人/事/物/景) |
| 第二步 | 抓意象，理解象征 | 找出诗中关键意象，联想其常见象征意义(月亮=思乡，柳=送别，梅=坚韧)，结合语境判断 |
| 第三步 | 析手法 | 识别修辞手法(比喻、拟人、夸张、对偶)和表现手法(借景抒情、托物言志、动静结合、虚实相生) |
| 第四步 | 悟情感 | 综合意象和手法，判断诗人情感(喜悦/忧伤/思乡/爱国/壮志/闲适)，用准确词语概括 |
