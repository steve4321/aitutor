## Coverage

### AMC (American Mathematics Competitions)
- AMC 8 / AMC 10 / AMC 12 / AIME
- Four pillars: Algebra, Geometry, Counting & Probability, Number Theory
- Dual mode: Course (learn foundations) + Practice (AMC problem solving)

### KET (Cambridge English: Key)
- CEFR A2 level
- Four skills: Reading, Writing, Listening, Speaking
- Cambridge official rubrics alignment
- Spaced repetition vocabulary system
